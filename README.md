
# Табачки 2.0 — Готовый проект
Подключено к твоему Apps Script (URL и API_KEY уже проставлены в lib/src/config.dart).

## Запуск
```
flutter pub get
flutter run
```
Сборка APK:
```
flutter build apk
```

## Примечание
- Если хочешь включить распознавание по фото — добавь ключ в `Config.visionApiKey`.
- Листы в Google Sheets должны соответствовать: Stores, Employees, Shifts, ZReports, Expenses, Incomes, Stock, Ratings, Notifications.
